Customer API Overview

General Information

This API is designed for managing customer data. Below are the available endpoints.

Available Endpoints

  • Home Route:
    • Endpoint: /
    • Method: GET
    • Description: Serves the home page.
  • Get All Customers:
    • Endpoint: /customers
    • Method: GET
    • Description: Retrieves all customer details in JSON format.
  • Get Single Customer:
    • Endpoint: /customers/{id}
    • Method: GET
    • Description: Retrieves details of a single customer based on the ID.
  • Create Single Customer:
    • Endpoint: /customers
    • Method: POST
    • Description: Creates a new customer with a unique ID.
  • Update Single Customer:
    • Endpoint: /customers/{id}
    • Method: PATCH
    • Description: Updates the details of an existing customer based on the ID.
  • Delete Single Customer:
    • Endpoint: /customers/{id}
    • Method: DELETE
    • Description: Deletes a customer based on the ID.

For further details and usage, please refer to the API documentation.
